## 🚀 Overview

**AI RSS SEO Generator** is a powerful WordPress plugin that automates your content creation workflow. It fetches articles from any RSS feed, rewrites them using **DeepSeek AI**, adds **2 high-quality images** from Pexels, embeds a **relevant YouTube video**, and optimizes everything for **SEO** - all automatically!

Perfect for:
- 📰 News websites
- ✍️ Bloggers
- 📈 SEO agencies
- 🛍️ Affiliate marketers
- 🗂️ Content curators

## ✨ Key Features

### 🤖 **AI Content Generation**
- Uses **DeepSeek API** (superior to ChatGPT for Persian content)
- Customizable prompts for titles and content
- Generates **1500+ word articles** with 10+ paragraphs
- Maintains your unique brand voice

### 🖼️ **Automatic Images**
- Fetches **2 high-quality images** from Pexels
- Strategic placement (paragraphs 3 and 9)
- First image becomes featured image
- **200 free requests/hour**

### 🎬 **YouTube Integration**
- Searches for most relevant video
- Embeds video in paragraph 6
- Supports Persian language search
- Fully responsive player

### 📈 **SEO Optimization**
- Generates SEO-friendly titles
- Creates **meta descriptions** (155 characters)
- Customizable title tags with `{title}` and `{site_name}`
- Compatible with **Yoast SEO, Rank Math, All in One SEO**
- Clean URL slugs from titles

### ⏰ **Cron Job Support**
- Built-in WordPress cron (hourly)
- Beautiful external cron URL: `https://yoursite.com/6arshid-cron/`
- Customizable post limits per run
- Perfect for **Hostinger hPanel**

### 📊 **Advanced Features**
- ✅ Duplicate post prevention
- ✅ Detailed logging system
- ✅ One-click manual generation
- ✅ Test connection button
- ✅ View logs in admin panel
